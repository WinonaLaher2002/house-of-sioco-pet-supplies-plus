# House of Sioco Pet Supplies Plus

A static website for House of Sioco Pet Supplies Plus, a neighborhood pet supply store in Veinte Reales, Valenzuela City.

The site includes sections for the shop story, product categories, customer reviews, contact details, store hours, map embed, and links to Shopee and Facebook.

## Project Files

```text
.
|-- index.html      Main website content and page structure
|-- styles.css      Website styling and responsive layout
|-- script.js       Mobile menu, sticky header, and dynamic footer year
|-- assets/         Logo and favicon files
```

## How to Open

This is a plain HTML/CSS/JavaScript website, so no install step is needed.

Open `index.html` directly in a browser, or use a local preview extension such as Live Server in VS Code.

## Common Edits

Edit `index.html` for:

- Store name and page text
- Product categories
- Reviews
- Address, phone number, email, and social links
- Shopee and Facebook URLs
- Store hours
- Map embed URL

Edit `styles.css` for:

- Colors
- Fonts
- Spacing
- Layout
- Mobile responsiveness

Edit `script.js` for:

- Mobile menu behavior
- Header scroll behavior
- Footer year behavior
